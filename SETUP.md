# KULTI — Getting Started

Guide for running the project for the first time.

## Prerequisites

- [Docker](https://docs.docker.com/get-docker/)
- [Python 3.12+](https://www.python.org/downloads/)
- [Node.js 18+](https://nodejs.org/)

## Setup Steps

1. **Environment variables** — copy and edit the env file:
   ```bash
   cp .env.example .env
   ```

2. **Database** — follow [docs/docker-setup.md](docs/docker-setup.md)

3. **Backend** — follow [docs/backend-setup.md](docs/backend-setup.md)
